Interface IPipeFitting

Pipe Fitting Interface.

An IPipeFitting< can be connected to other IPipeFittings, forming a Pipeline. Messagess are written to one end of a Pipeline by some client code. The messages are then transferred in synchronous fashion from one fitting to the next.

interface IPipeFitting {
    connect(output: IPipeFitting): boolean;
    disconnect(): undefined | IPipeFitting;
    write(message: IPipeMessage): boolean;
}

write

  • write(message: IPipeMessage): boolean

    Write the message to the output Pipe Fitting. There may be subsequent filters and tees (which also implement this interface), that the fitting is writing to, and so a message may branch and arrive in different forms at different endpoints. If any fitting in the chain returns false from this method, then the client who originally wrote into the pipes.ts can take action, such as rolling back changes.
